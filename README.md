# MindToCode

**Technical Guidance & Digital Solutions Platform**

A premium, single-page application for converting learners and businesses into clients.

## 🚀 Project Overview
- **Zero Libraries:** 100% Custom Components
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS + Custom CSS Modules
- **Database:** Supabase (PostgreSQL)
- **Email:** Resend
- **Auth:** Custom Admin System

## 🛠️ Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Set up environment variables:**
   - Copy `.env.example` to `.env.local`
   - Fill in your Supabase and Resend keys

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open [http://localhost:3000](http://localhost:3000)**

## 📂 Project Structure
- `/app` - Routes and pages
- `/components` - Reusable custom components
- `/lib` - Utilities and database clients
- `/styles` - Global styles and CSS modules
- `/hooks` - Custom React hooks
